# Threat Model — A-DAP

This document defines the threat model for A-DAP — Auditable Decision Accountability Protocol.

A-DAP is a verifiability architecture for externally reconstructible automated decisions.

Its purpose is not to eliminate all possible trust, risk, or institutional failure.

Its purpose is to identify where auditability fails, where reconstruction is possible, where evidence depends on self-attestation, and where a boundary must be explicitly declared.

A-DAP focuses on this question:

```text
Can an external reviewer reconstruct what must be true for the decision to be trusted?
