## 1. Purpose

This document defines a strict 90-day decision framework for A-DAP / ReconBench.

Its purpose is not to prove that the project is important.

Its purpose is to test whether the project deserves continued strategic energy.

The central question is:

> Does anyone outside the maintainer’s own circle have enough interest to reconstruct, attack, critique, fork, cite, or request the continuation of this work?

If the answer is no, the project should not continue expanding as if demand exists.
