# GCD-001 Submission Template

This file provides the recommended submission format for external reviewers submitting a result for the GCD-001 External Reconstruction Package.

Submissions should classify their result as one of three admissible outcomes:

- **R1 — Reproduction**
- **R2 — Falsification**
- **R3 — Structural Ambiguity**

At the current stage, formal submission is not yet open.

This template defines the intended submission structure for future external review.
